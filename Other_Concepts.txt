----------------------------------------------------------------------------
VARIABLE HOISTING
----------------------------------------------------------------------------

console.log(a);
var a = 20; 

The output is undefined, as in this case the variable declaration will be 
copied to top of it's scope and not declaration

It can be done with 'var' keyword and not with 'let' and 'const'

----------------------------------------------------------------------------
FUNCTION HOISTING
----------------------------------------------------------------------------

greet();

function greet(){
    console.log("Hello");
}

It'll give the output as Hello, since the complete function definition will
comes to the top of the scope

This can't be done with the function expression

----------------------------------------------------------------------------

See the below code for better understanding:

}

A(){
